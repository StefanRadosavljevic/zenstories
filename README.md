## Install

```sh
flutter pub get
```

## Configuration

Open the `constants.dart` file inside `lib` folder.

```dart
const Map<String, String> Constants = const {
  'API_BASE_URL': '<ROOT_URL>/wp-json/wp/v2'
};
```

Replace ROOT_URL with your wordpress root URL.

## Usage

```sh
flutter run
```

## Run tests

```sh
flutter test
```
